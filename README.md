# Порівняльний аналіз архітектурних патернів для iOS-застосунків

## Про проєкт

Дослідницький проєкт студентського наукового гуртка "Теорія та практика програмної інженерії" кафедри КІТ ННІ ДІІТ УДУНТ.

**Мета:** розробити науково обґрунтовану порівняльну оцінку основних архітектурних патернів для iOS-розробки на основі систематичного огляду літератури за методологією PRISMA та емпіричного дослідження з використанням кількісних метрик якості коду.
